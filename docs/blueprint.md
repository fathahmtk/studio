# **App Name**: Culinary Cost Manager

## Core Features:

- Ingredient Database: Maintain a database of ingredients with supplier details, purchase costs, and unit measurements to accurately track ingredient expenses.
- Recipe Builder: Automatically calculate the total ingredient cost for each menu item based on the recipe and current ingredient prices.
- Labor and Overhead Calculator: Add labor costs and overhead expenses to the menu item's ingredient cost to calculate a final menu price that ensures profitability.
- Menu Cost Analysis: Generate comprehensive reports on menu item costs, profitability margins, and identify areas for cost optimization. Report on the costs of different portion sizes.
- Supplier Integration: Analyze costs across different suppliers using AI tools.

## Style Guidelines:

- Primary color: Forest Green (#388E3C) to evoke freshness and financial health.
- Background color: Light Silver (#D3D3D3) for a clean, professional look.
- Accent color: Teal Green (#008080) to complement the primary color and add depth.
- Body and headline font: 'Inter', a sans-serif font providing a clean and professional feel for both headings and body text.
- Use minimalist icons representing ingredients, cost analysis, and financial data. These should align with the professional aesthetic.
- Ensure a clean and intuitive layout for ease of navigation. Prioritize clear data presentation with appropriate spacing.
- Subtle transitions and animations to provide a smooth user experience when navigating or displaying data. Avoid distracting animations.
